## Legenda colonne
- **Folder** = 24-char hex ID in `I:\BKS database\`
- **bp** = Printify blueprint_id (intero) usato nella API
- **AI/PSD** = template vettoriale editabile
- **PNG** = template PNG pronti per OpenAI images.edit
- **Aree PNG** = print area positions con template PNG disponibile

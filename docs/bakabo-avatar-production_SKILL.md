# bakabo-avatar-production

Guida operativa per produrre video avatar BakAbo/BKS su HeyGen.

## Trigger

Usala quando il team chiede script avatar, video collection, TikTok/Reel parlato,
configurazione HeyGen, export MP4 o delivery metadata per una collection BKS.

## Workflow

1. Brief: collection, canale, CTA, visual di partenza.
2. Script: 15 secondi, 35-50 parole, voce calma, nessun punto esclamativo.
3. Image: scegli una hero image e portala a 9:16.
4. HeyGen: avatar approvato, brand voice, script, immagine, export.
5. Delivery: MP4, QC checklist, metadata.

## Regole script

- Prima frase: BKS + nome collection.
- Una sola idea per frase.
- Nessun hard sell.
- Nessun punto esclamativo.
- CTA finale semplice: explore, save, follow, test the drop.

## Immagini

Sorgenti consigliate:

- `output/collection_image_program/curated/contact_sheets/`
- `output/collection_image_program/contact_sheets/`
- `02_COLLEZIONI/`

Naming finale:

```text
output/avatar_production/images/BKS_[COLLECTION]_hero_[DATE].jpg
```

## HeyGen

1. Crea video portrait.
2. Seleziona avatar BakAbo approvato.
3. Seleziona brand voice.
4. Incolla script finale.
5. Carica immagine 9:16.
6. Esporta MP4.
7. Esegui QC.

## Delivery

Salva tutto in:

```text
output/avatar_production/
```

Cartelle standard:

- `scripts/`
- `images/`
- `exports/`
- `metadata/`

Canali previsti:

- Homepage
- Instagram Reels
- TikTok
- YouTube Shorts
- Email

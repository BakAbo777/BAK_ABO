# PARTE 2 — AUDIT METAFIELD LEGACY

Dalla schermata BKS Hours emergono i seguenti **metafield legacy da rimuovere**:

| Metafield (etichetta) | Problema | Azione |
|---|---|---|
| `BKS 🏄 Pullover Hoodie` | Emoji + legacy label format | Eliminare definizione |
| `BKS 👟 Line 1 Sneakers` | Emoji + "Line 1" è naming legacy | Eliminare |
| `BKS 🎒 Backpack` | Emoji in metafield label | Eliminare |
| `BKS 🎯 Flip Flops` | Emoji + "Flip Flops" vs handle `flip-flop` | Eliminare |
| `BKS 💼 Duffel Bag` | "Duffel Bag" — non corrisponde ad alcun tipo prodotto BKS attivo | Eliminare |
| `BKS 🧥 Windbreaker Jackets` | Emoji + "Jackets" (plurale non standard) | Eliminare |
| `BKS Line 2 👟 Sneakers` | Legacy "Line 2" naming | Eliminare |
| `BKS 🏖️ Beach towels` | Categoria non attiva nel catalogo 2026 | Eliminare |
| `BKS 🏝️ Island Sneakers` | Legacy nome linea | Eliminare |
| `BKS 👑 Tiaras & Stamps...` | Legacy prodotto dismesso | Eliminare |
| `BKS 🇯🇵 Japan Sneakers` | Bandiera nazione (viola guardrail Flag) + legacy | Eliminare |
| `BKS 🍊 Mondello Sneakers` | Legacy nome luogo | Eliminare |
| `BKS 🧥 Puffer Jacket` | Emoji (duplica tipo prodotto attivo, ma con emoji) | Eliminare |
| `BKS 🥿 Indoor Slippers` | Emoji + "Indoor Slippers" vs `cozy-slipper` | Eliminare |
| `BKS 🕹️ Arcade Sneakers` | Emoji + legacy naming | Eliminare |
| `BKS 🐟 Fish Citizens...` | Legacy nome prodotto | Eliminare |
| `BKS 🏊 Swim Trunks` | Emoji | Eliminare |
| `BKS 💃 Chic One-Piece...` | Emoji + "Chic" è voce legacy | Eliminare |

**Azione:** Admin Shopify → Settings → Custom data → Metafields → Collections → eliminare tutte le definizioni con emoji o naming legacy. Le definizioni valide del sistema 2026 sono le 10 definite in `create_metafields.py` (bks.collection, bks.design, bks.series, bks.drop, ecc.).

# BakAbo — Art Critic (Graphics only)

`bakabo-art-critic` — Reads and writes about BakAbo GRAPHICS (prints, marks, motifs, chromatic fields) as visual art objects. Scope: the graphic layer only — what is printed, what it enacts, what it proposes. Not fabric, not sizing, not shipping. Those are handled by `bks-product-editorial-care` and Shopify. This skill produces gallery card reviews, collection press notes, extended editorial essays, and review prompts that elevate the graphic as cultural artifact.

## What this skill does differently

| Standard copy | Art critic |
| --- | --- |
| "Bold graphic print" | "The print enacts a visual frequency — repeating motif at irregular intervals, creating rhythm without resolution" |
| "Limited edition" | "One iteration of an ongoing series — each drop is a published chapter, not a clearance" |
| "Inspired by music" | "The collection is a score — color becomes pitch, repeat becomes beat, the garment is a notation you wear" |

## Output formats

**Gallery card (50–120 words):** One sentence each — what the object IS formally, what the print DOES, why the material matters, where in the BKS series this sits. Ends with: `Collection: BKS {Name} | Typology: Made-to-order | Series: Drop N`

**Press note (150–250 words):** Collection introduction as if at a gallery opening. Structure: the argument → formal response → cultural moment → invitation (never a call to purchase).

**Critical essay (500–800 words):** Opening concrete image → formal analysis → conceptual frame → cultural placement → close that returns to the opening image.

**Customer review prompts:** Replace "what did you think?" with questions that produce specific, usable, editorial-quality reviews.

## Critical vocabulary (key terms)

Register, ground, figure/ground relationship, chromatic field, motif, palette (as compositional decision), scale, iteration, drop, notation, surface.

**Never:** "beautiful", "stunning", "luxurious" without specifics, urgency language, emoji.

## Reading a BKS product — 6-question sequence

1. What is the mark? (print type: geometric, typographic, abstract)
2. What is the field? (fabric quality, color, weight)
3. How do mark and field interact? (neutral / tension / harmony)
4. What is the collection's musical reference? (how does visual encode sound?)
5. What is the scale? (print size vs. human body — monumental, intimate, deliberate tension)
6. What iteration is this? (what changed from previous drop?)

## Brand voice compatibility

Compatible with `bakabo-brand` because both: refuse urgency, refuse exclamation marks, trust the reader, are specific not vague. Difference: brand voice = studio speaking; critic = independent reading. Use critic when the product needs to be READ as art, not just sold as fashion.

# BKS / BakAbo — Product Naming Rules

## Standard structure

```
BKS [Collection] [Product Type] — [Variant Name]
```

## Rules

1. **Always start with BKS** — never omit the brand prefix
2. **Collection name** — one of the 8 official collections (Hours/Origin/Glyph/Marker/Riviera/Pulse/Token/Flag)
3. **Product Type** — descriptive English noun (Tee, Puffer Jacket, Windbreaker, Sneakers, etc.)
4. **Variant name** — short evocative word (1–2 words), connected to collection DNA
5. **Separator** — em dash (—) between product type and variant name

## Examples (correct)

```
BKS Glyph Sneakers — Shade
BKS Riviera One-Piece Swimsuit — Coast
BKS Flag Windbreaker — Manifesto
BKS Hours Lounge Pants — Pane
BKS Marker Athletic Long Shorts — City
BKS Pulse Leggings — Wave
BKS Token Backpack — Signal
BKS Origin Hoodie — Earth
```

## Forbidden patterns

| Pattern | Example | Why |
|---------|---------|-----|
| Emoji in title | "BKS Glyph Tee 🖤" | Cheapens SEO + brand |
| "Copy of" prefix | "Copy of BKS Hours Tee" | Printify artifact — clean before publish |
| Missing collection | "BKS Sneakers" | Not assignable to collection family |
| All caps | "BKS GLYPH TEE" | Not editorial |
| Numbers as names | "BKS Tee 001" | Generic |
| Extra long titles | "BKS Glyph Cross Symbol Pattern All Over Print Premium..." | SEO garbage |
| Mixed language | "BKS Glyph Maglietta" | English only for product names |
| POD blueprint names | "Men's Bomber Jacket Glyph" | Strip blueprint name from title |

## Variant name inspiration

| Collection | Good variant names |
|-----------|-------------------|
| Hours | Pane, Grid, Ink, Silence, Arc |
| Origin | Root, Clay, Earth, Moss, Layer |
| Glyph | Shade, Mark, Cross, Script, Sign |
| Marker | City, Code, Signal, Trace, Tag |
| Riviera | Coast, Lido, Onda, Maritime, Bay |
| Pulse | Wave, Beat, Signal, Frequency, Arc |
| Token | Hash, Block, Chain, Key, Signal |
| Flag | Manifesto, Field, Drop, Burst, Form |

## Title length

- Maximum: 60 characters (Shopify SEO limit)
- Target: 35–55 characters
- Minimum: 20 characters

## Shopify handle

Auto-generated from title. Never use emoji or special characters in handles.

## ™ mark (internal)

Products with ™ in the Printify title are approved for AI design generation. This mark is internal and should NOT appear in the final Shopify product title.

# BKS / BakAbo — Ad Voice & Copy Rules

## Voice profile

| Attribute | Description |
|-----------|-------------|
| Tone | Raw editorial. Direct. Slightly poetic. |
| Energy | Calm confidence, not aggressive hype |
| Geography | Urban, architectural, Italian-rooted |
| Authenticity | Honest about POD production, proud of AI design |
| Complexity | Short sentences. No jargon. No fluff. |

## What to always include in ads

1. Strong image (product readable)
2. Clear product identity
3. Credible promise
4. Made-on-demand mention
5. Short CTA

---

## Hard rules — NEVER write

| Forbidden | Why |
|-----------|-----|
| "luxury handcrafted" | False — POD production |
| "best quality ever" | Unverifiable, generic |
| "limited edition" | Only if drop is truly limited |
| "Italian made" | Not manufactured in Italy |
| "premium materials" | Only if declared and verifiable |
| "handmade in [country]" | Not handmade |
| Excessive emoji | Cheapens brand |
| Dropshipping-style copy | Destroys brand trust |
| Fake urgency ("only 3 left!") | Not applicable to POD |

# BKS — No Fake Luxury Rules

## Core principle

BakAbo is a made-on-demand brand. Every garment is produced after the order by a global print partner.
Never describe it as handmade, artisanal, Italian-manufactured, or traditionally luxury.

Raise perception through: visual system, editorial images, collections, honest naming, and AI design quality.

---

## Forbidden phrases (complete list)

| Forbidden | Correct alternative |
|-----------|-------------------|
| "luxury handcrafted" | "raw editorial artwear" |
| "Italian made" | "designed in Italy" |
| "hand-sewn by artisans" | "made on demand worldwide" |
| "exclusive collection" | "curated collection" |
| "limited edition" | omit unless drop is truly limited |
| "best quality ever" | specify material (e.g. "100% ringspun cotton") |
| "premium materials" | declare actual material |
| "only 3 left" | not applicable — POD has no real stock |
| "artisanal production" | "AI-generated, made on demand" |
| "haute couture" | not applicable |
| "high-end fashion brand" | "AI-art fashion atelier" |
| "ultra-premium" | no superlatives without evidence |
| "finest fabrics" | declare actual fabric |
| "made with love in Italy" | "designed in Italy" |

---

## What raises perception correctly

1. **Visual system** — editorial images, golden ratio composition, clean product shots
2. **Naming** — BKS [Collection] [Type] — [Variant] — no emoji, no copy-of
3. **Collections** — 8 named families with clear visual DNA
4. **AI design** — explain the AI-generated pattern, not fake the artisan story
5. **Copy** — honest about production, proud of design
6. **Price** — only supported if image + copy + material are all in order

---

## The real BakAbo promise

```
AI-generated pattern designed by BKS Studio.
Applied to a [product type] made on demand after your purchase.
No overstock. Designed in Italy. Produced worldwide.
```

---

## When to use "premium"

Only say "premium" when:
- The product material is declared and above average
- The image is editorial quality
- The copy is clean and honest
- The price reflects added value from design, not inflated POD margin

Never use "premium" to compensate for weak imagery or generic description.

---

## Internal quality ladder

```
Tier 1 (avoid): Generic Printify description + mockup JPG
Tier 2 (minimum): Product name + material + basic copy
Tier 3 (target): BKS naming + editorial image + AI design story + material + fit
Tier 4 (ideal): All of Tier 3 + collection context + made-on-demand explanation + schema markup
```

---

## Tone test

Before publishing any copy, ask:
- Does this sound like a real editorial fashion brand?
- Could a customer who reads this feel deceived about how the product is made?
- Am I claiming something I cannot prove?

If any answer is "yes" → rewrite before publishing.

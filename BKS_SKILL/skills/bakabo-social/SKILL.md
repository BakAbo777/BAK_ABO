## 4. YouTube — formati e specifiche tecniche

### Formati video per upload

| Tipo | Risoluzione | Ratio | Durata | Uso |
|---|---|---|---|---|
| Video standard | 1920×1080 | 16:9 | 1–5 min | Editoriale collezione, processo AI |
| YouTube Short | 1080×1920 | 9:16 | max 60s | Clip prodotto, pattern reveal |
| Thumbnail | 1280×720 | 16:9 | — | Obbligatoria custom per ogni video |

### Codec raccomandati
- Container: MP4
- Video codec: H.264 o H.265
- Audio: AAC 44.1kHz stereo
- Bitrate video: min 8 Mbps per 1080p
- Frame rate: 24fps (editoriale) o 30fps (prodotto)

### Nomenclatura file prima dell'upload
```
bks-[tipo]-[collezione]-[design]-[numero].mp4
```
Esempi:
- `bks-short-marker-fawn-puffer-01.mp4`
- `bks-editorial-glyph-collection-hero-01.mp4`
- `bks-reel-pulse-arc-onepiece-01.mp4`

## 12. YouTube Data API v3 — automazione upload e metadata

Per automatizzare upload e aggiornamento metadata video senza accesso manuale a YouTube Studio.

### Setup (una tantum)
1. Google Cloud Console → nuovo progetto "BKS Studio YouTube"
2. Abilita YouTube Data API v3
3. Credenziali → OAuth 2.0 → Desktop App → scarica `client_secret.json`
4. Prima esecuzione: autorizza via browser → genera `token.json` (riutilizzato automaticamente)

### Script Python — aggiornamento bulk metadata video esistenti

```python
# bks_youtube_update.py
# Aggiorna titolo, descrizione, tag di tutti i video del canale

import os
import json
from google.oauth2.credentials import Credentials
from google_auth_oauthlib.flow import InstalledAppFlow
from google.auth.transport.requests import Request
from googleapiclient.discovery import build

SCOPES = ["https://www.googleapis.com/auth/youtube.force-ssl"]
CLIENT_SECRETS = "client_secret.json"
TOKEN_FILE = "token.json"

def get_authenticated_service():
    creds = None
    if os.path.exists(TOKEN_FILE):
        creds = Credentials.from_authorized_user_file(TOKEN_FILE, SCOPES)
    if not creds or not creds.valid:
        if creds and creds.expired and creds.refresh_token:
            creds.refresh(Request())
        else:
            flow = InstalledAppFlow.from_client_secrets_file(CLIENT_SECRETS, SCOPES)
            creds = flow.run_local_server(port=0)
        with open(TOKEN_FILE, "w") as f:
            f.write(creds.to_json())
    return build("youtube", "v3", credentials=creds)

def update_video_metadata(youtube, video_id: str, title: str, description: str, tags: list):
    youtube.videos().update(
        part="snippet",
        body={
            "id": video_id,
            "snippet": {
                "title": title,
                "description": description,
                "tags": tags,
                "categoryId": "26"  # 26 = How-to & Style
            }
        }
    ).execute()
    print(f"Updated: {video_id} — {title}")

def list_channel_videos(youtube) -> list:
    """Restituisce lista di tutti i video del canale autenticato."""
    videos = []
    request = youtube.search().list(
        part="snippet",
        forMine=True,
        type="video",
        maxResults=50
    )
    while request:
        response = request.execute()
        for item in response.get("items", []):
            videos.append({
                "id": item["id"]["videoId"],
                "title": item["snippet"]["title"],
                "published": item["snippet"]["publishedAt"]
            })
        request = youtube.search().list_next(request, response)
    return videos

# Esempio: aggiorna un video specifico
if __name__ == "__main__":
    yt = get_authenticated_service()
    
    # Elenco video da aggiornare: {video_id: {title, description, tags}}
    updates = {
        "UXKXD4lzTuM": {
            "title": "BKS Marker Collection — AI Pattern Editorial",
            "description": (
                "An editorial exploration of the BKS Marker collection.\n"
                "AI-generated patterns in the neo-expressionist system — "
                "gestural marks, urban surfaces, colour as signal.\n\n"
                "BKS Studio — bakabo.club\n"
                "Contact: crew@bakabo.club\n\n"
                "#BKSStudio #AIArtFashion #WearableArt #MarkerCollection"
            ),
            "tags": [
                "BKS Studio", "AI art fashion", "wearable art",
                "print on demand", "BKS Marker", "puffer jacket",
                "AI generated clothing", "Italian design", "editorial fashion"
            ]
        }
    }
    
    for vid_id, meta in updates.items():
        update_video_metadata(yt, vid_id, meta["title"], meta["description"], meta["tags"])
```

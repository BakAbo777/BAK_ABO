# BKS BRAND POSITIONING — No-Luxury Rule

## Regola fondamentale

**BakAbo / BKS Studio NON è un luxury brand.**

Non descrivere BakAbo come: luxury brand, alta moda, sartoria, handmade luxury, premium luxury,
high-end fashion, boutique luxury, lusso italiano, luxury streetwear, affordable luxury.

Se una proposta "sembra luxury finto" → BKS Score automaticamente < 15/25.

---

## Formula brand ufficiale

**EN:**
> BakAbo / BKS Studio is an AI-art fashion atelier designed in Italy and made on demand worldwide. The brand creates AI-generated all-over print surfaces, curated as editorial artwear and applied to everyday garments and accessories.

**IT:**
> BakAbo / BKS Studio è un atelier di AI-art fashion progettato in Italia e prodotto on demand nel mondo. Il brand crea superfici all-over print generate con AI, curate come artwear editoriale e applicate a capi e accessori d'uso quotidiano.

**SHORT (ads / social):**
> Raw editorial artwear. Designed in Italy, made on demand worldwide.

## Il punto forte BakAbo

Il valore non è il capo base Printify. Il valore è:
1. Sistema visivo BKS (8 collezioni)
2. Superfici AI-generated originali proprietarie
3. Curatela editoriale da studio creativo italiano
4. Prodotto on-demand senza magazzino
5. Set fotografici architettonici BKS

Il cliente non compra "una felpa stampata" — compra un pezzo dentro un universo visivo coerente.

## Struttura scheda prodotto (5 blocchi fissi)

```
1. TITOLO
   BKS [Collection] [Design]™ [Tipo Prodotto] — max 60 caratteri

2. HERO LINE (editoriale, una frase in corsivo)
   design → materiale → vita del capo
   es: "A wide-leg lounge pant carrying a fragmented BKS Glyph field in lavender and slate grey."

3. VERITÀ PRODOTTO (onestà obbligatoria)
   "Made on demand. Printed after purchase." + materiale reale (polyester, cotton, gsm)
   MAI ingannare il cliente sulla natura del prodotto.

4. VALORE BKS (colloca nel sistema visivo)
   "Part of BKS [Collection]: AI-generated [elemento design] curated by BakAbo Studio
    as wearable [descrittore]."

5. FIDUCIA (fisso, non modificare mai)
   "30-day returns · EU 2-year warranty · production before shipping."

SPEC BLOCK (separato): Materiale · Fit/Capacità · Stampa · Cura
```

## BKS Score 1/25 — formula valutazione

```
BKS Score = (BKS Identity Fit    × 0.15)
          + (Visual Strength      × 0.15)
          + (Product Truth        × 0.15)
          + (Commercial Clarity   × 0.15)
          + (Mobile Impact        × 0.10)
          + (Trend Compatibility  × 0.15)
          + (Color/Garment Fit    × 0.10)
          + (Risk Safety          × 0.05)
```

### Criteri

1. **BKS Identity Fit** — quanto è riconoscibile come BakAbo / BKS?
2. **Visual Strength** — quanto è forte visivamente?
3. **Product Truth** — quanto è credibile rispetto a Printify / made-on-demand?
4. **Commercial Clarity** — il cliente capisce cosa compra?
5. **Mobile Impact** — funziona in 3 secondi da cellulare?
6. **Trend Compatibility** — coerente con trend attuali o futuri?
7. **Color / Garment Fit** — colori e capo adatti alla fascia d'età scelta?
8. **Risk Safety** — evita falso lusso, estetiche violente, promesse non verificabili?

### Scala decisionale

| Score | Decisione |
|---|---|
| 1–5 | IGNORE |
| 6–10 | WATCH |
| 11–15 | TEST |
| 16–19 | PUSH LIGHT |
| 20–22 | PUSH |
| 23–25 | BUILD COLLECTION |

### Soglie operative

- ≥ 16/25 → valida per test in prodotto o contenuto
- ≥ 20/25 → valida per campagna
- ≥ 23/25 → valida come capsule o direzione di collezione
- Se "sembra luxury finto" → abbassare sotto 15 automaticamente

# BKS Image Pipeline Skill

## What it does
Downloads Printify product mockups, removes backgrounds (rembg conservative mode), generates 10 overlays per product (8 BKS collection accents + salt + onyx), and optionally uploads the clean version to Shopify.

## Primary script
`BAKABO_IMAGE_FACTORY_v1.1/scripts/printify_cutout_pipeline.py`

## Python environment
Run with `.venv_dashboard` (has onnxruntime + rembg):
```
.venv_dashboard\Scripts\python.exe scripts/printify_cutout_pipeline.py
```

## CLI usage
```bash
# Test 1 product (no upload)
.venv_dashboard\Scripts\python.exe scripts/printify_cutout_pipeline.py --max 1

# Process all products
.venv_dashboard\Scripts\python.exe scripts/printify_cutout_pipeline.py

# Process + upload clean version to Shopify
.venv_dashboard\Scripts\python.exe scripts/printify_cutout_pipeline.py --upload

# Force reprocess even if cached
.venv_dashboard\Scripts\python.exe scripts/printify_cutout_pipeline.py --force

# Full run with upload
.venv_dashboard\Scripts\python.exe scripts/printify_cutout_pipeline.py --upload --max 10
```

## Outputs per product
| File | Description |
|------|-------------|
| `{slug}_cutout_safe.png` | RGBA transparent cutout (Shopify-ready) |
| `{slug}_transparent.png` | Same as cutout_safe (alias) |
| `{slug}_salt_commercial_bg.jpg` | Product on #FAFAF7 salt paper |
| `{slug}_onyx_overlay.jpg` | Product on #0A0A0A cinema dark |
| `{slug}_{collection}_overlay.jpg` | One per BKS accent (8 files: hours/glyph/marker/riviera/pulse/token/flag/origin) |
| `{slug}_edge_review.jpg` | QA: split-BG edge check |
| `{slug}_checkerboard_review.jpg` | QA: transparency checkerboard |
| `{slug}_alpha_mask_review.png` | QA: alpha channel mask |

## Background removal settings (conservative mode)
- Model: `u2net` (rembg)
- `alpha_matting=True`
- `foreground_threshold=250` — keeps more garment (less erosion)
- `background_threshold=5` — less BG expansion
- `erode_size=3` — small erode, preserves edges
- Edge expansion enabled by default → no eaten seams

## Shopify upload (`--upload` flag)
- Uploads `_salt_commercial_bg.jpg` to Shopify as position=2 image
- Requires product to have Shopify ID in Printify `external.id` field
- Only works if the product is already published to Shopify

## Key modules
| Module | Description |
|--------|-------------|
| `modules/background_remover.py` | BG removal + overlay composition (CANONICAL) |
| `modules/shopify_client.py` | Shopify image upload + product update |
| `modules/printify_client.py` | Printify product list + shop resolution |
| `modules/image_analyzer.py` | Quality analysis + edge metrics |

## Pre-publish gate
Before uploading any image to Shopify, run:
1. Armocromista: check accent color match + contrast
2. Photo Studio: verify framing, ratio, BKS composition rules
3. SEO: check alt text (generated by `modules/seo_generator.py`)

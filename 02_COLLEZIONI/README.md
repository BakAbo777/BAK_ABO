# Fase 02 — Collezioni

Obiettivo: creare/sincronizzare le 25 collection BKS, template assignment, SEO collection e prompt immagini.

Launcher:

```bat
02_START_COLLECTIONS_DASHBOARD.bat
```

Usare nel cruscotto:

- `01 Ricarica` per controlli CSV e metafield preliminari.
- `02 Piano V20` per export collection/template.
- `03 Collection` per sync Shopify.
- `05 Immagini AI` per prompt e asset collection.
- `06 Analytics` per link, GTM, GA4 e Merchant Center.

Output principali:

- `output/bks_collection_plan_v20.csv`
- `output/bks_collection_template_assignment_v20.csv`
- `output/bks_collection_payloads_v20.json`
- `output/bks_collection_image_prompts_v20.md`
